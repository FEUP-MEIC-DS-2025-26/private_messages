# Contribution guidelines

Thank you in advance for your interest in improving this microservice!
If you have any questions regarding our group, feel free to write in the GitHub Discussions or send a direct message the Scrum Master in Microsoft Teams:
- João Vítor da Costa Ferreira (up202208393@up.pt)

## Proposing an endpoint to integrate with other prototype
Create a GitHub issue with the Endpoint template and fill the respective fields.
It the endpoint is urgent, feel free to ping the Scrum Master in Microsoft Teams.

## Proposing a suggestion, feature or bug
Feel free to create a GitHub Issue detailing what needs to be done.
There is not a specific format that the issue needs to have

## Develop code for an issue
Feel free to create a pull request from a fork or a branch to main.
To notify us about the PR, place the Scrum Master as a reviewer of the PR.

## Notify about a security vulnerability
Check [SECURITY.md](./SECURITY.md).
